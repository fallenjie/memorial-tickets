## 功能需求

### 2.1 票根上传
- 支持拍照、相册选择、拖拽上传三种方式
- 上传后进入**裁剪流程**，支持 **2:1** 或 **3:2** 两种比例，**由系统根据图片自动判断最优比例**，用户也可手动切换比例重新裁剪
- 裁剪完成后存储至 **IndexedDB**，保存原图 + 缩略图两个资源
- 压缩参数：**JPEG quality 0.8-0.85**，单张原图大小控制在 **2-3MB**

### 2.2 A 复古风格渲染
- 泛黄纸张纹理背景（**基础底色 #F0E2C0 + 随机微调**，每张票根有轻微色差），CSS gradient + noise 实现
- 撕裂边缘效果（CSS clip-path + SVG mask 实现，撕裂程度固定适中参数）
- 手写字体：ZCOOL KuaiLe / Ma Shan Zheng（CDN 加载）
- 票根打孔装饰（SVG circle holes）
- 褶皱纹理叠加（CSS 实现）

### 2.3 情感备注
- **选填**，上传流程中可跳过，也可上传后随时补填
- 一句话回忆：**140字以内**，文本输入
- 心情标签：**开心 / 感动 / 兴奋 / 平静 / 遗憾**（单选）
- 观演人数：**单人 / 双人 / 三人 / 四人+ / 家人**（单选）
- **可编辑**：保存后可再次编辑修改

### 2.4 票夹管理
- 列表展示：**按时间倒序**排列，卡片式布局
- **按类型筛选**：电影 / 演出 / 其他（**上传时由用户选择**）
- 搜索功能：支持按票根名称、备注内容关键词搜索

### 2.5 Export 数据导出
- 导出格式：JSON（元数据）+ 图片（ZIP 压缩包）
- 用途：**换手机/换浏览器数据迁移**
- **仅 Export，不需要 Import**（用户自行保管备份文件即可）

## 用户故事
- 作为用户，我希望上传实体票根照片并裁剪保存，这样即使票根丢失也能保留珍贵回忆
- 作为用户，我希望用复古风格渲染票根，让它们看起来有年代感和温度
- 作为用户，我希望为每张票根写下心情备注和观演人数，完整记录当时的场景
- 作为用户，我希望按类型筛选和搜索我的票夹，轻松找到特定回忆
- 作为用户，我希望导出全部数据到 ZIP，在换设备时能完整迁移
